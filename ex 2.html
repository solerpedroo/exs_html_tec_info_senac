Grasshopper é um aplicativo de aprendizado de programação com foco em JavaScript. O software é gratuito e está disponível para Android e iOS. Para facilitar os exercícios, ele traz um sistema de editor de códigos que basta arrastar e soltar, além de uma interface amigável; bem prático para quem está começando. Com este app, você aprende o básico de JavaScript, como funções, variáveis e métodos.
