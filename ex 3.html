Server Side vs. Client Side

Lado do Servidor (Server Side)

O código Server Side é executado no servidor antes que a página seja enviada para o navegador do usuário.

  • Exemplos de linguagens Server Side:
    • PHP
    • C#
    • Java
    • Python
  • Funcionalidades:
    • Processamento de dados no servidor
    • Acesso a bancos de dados
    • Geração dinâmica de conteúdo (por exemplo, páginas da web personalizadas)
    • Autenticação e autorização

Exemplo de Interação:

  1. Usuário → Cliente: O usuário entra com o endereço [URL] em seu navegador.
  2. Cliente → Servidor: O cliente realiza a requisição da página ao servidor de [URL].
  3. Servidor → Cliente: O servidor recebe a requisição, obtém a página e envia ao cliente.
  4. Cliente → Usuário: O cliente renderiza a página (HTML) recebida pelo servidor e exibe ao usuário.

Lado do Cliente (Client Side)

O código Client Side é executado no navegador do usuário, após o recebimento da página do servidor.

  • Exemplos de linguagens Client Side:
    • JavaScript
  • Funcionalidades:
    • Manipulação do DOM (Document Object Model)
    • Validação de formulários
    • Animações e interações na interface do usuário
    • Requisições assíncronas (AJAX)

Exemplo de Interação:

  1. Usuário → Cliente: O usuário entra com uma pesquisa no campo de pesquisas da página e clica em "Pesquisar".
  2. Cliente → Servidor: O cliente envia a requisição com o termo de busca informado pelo usuário.
  3. Servidor → Cliente: O servidor monta a página com os resultados da pesquisa e envia para o cliente.
  4. Cliente → Usuário: O cliente, ao receber do servidor a página com os resultados da pesquisa, renderiza a nova página e exibe para o usuário.
